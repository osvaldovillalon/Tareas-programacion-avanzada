##  Descripción

Esta carpeta contiene la solución completa a la **Actividad de Árboles de Decisión y Naive Bayes**.

La actividad consiste en:
- Aplicar el algoritmo ID3 (árbol de decisión) al dataset Iris.
- Utilizar Weka para evaluar el algoritmo J48.
- Implementar Naive Bayes en R para clasificar el dataset Iris.
- Evaluar el modelo con una matriz de confusión de 20 observaciones.
- Aplicar Naive Bayes a un conjunto de datos de estudiantes de la FCI.
